```
┌─────────────────────────────────────────────────────────────────────┐
│                        SYSTEM OF LAW                                 │
├─────────────────────────────────────────────────────────────────────┤
│  CONTINENTAL vs ANGLO-SAXON                                          │
│  ─────────────────────────────                                       │
│  Continental: Codes rule, judges apply                               │
│  Anglo-Saxon: JUDGES' DECISIONS = LAW (precedent)                    │
│                                                                      │
│  MONIST vs DUALIST                                                   │
│  ─────────────────                                                   │
│  Monist: ONE system (intl + domestic)                                │
│  Dualist: TWO separate systems                                       │
│  Poland: International > Statutes (priority)                         │
│                                                                      │
│  NATURAL vs POSITIVE LAW                                             │
│  ──────────────────────                                              │
│  Natural: Morality-based, precedes human law                         │
│  Positive: Human-made (statutes, codes)                              │
│  Living Constitution: Current meaning (opposite of originalism)      │
│                                                                      │
│  INTERPRETATION                                                      │
│  ─────────────                                                       │
│  Civil Law → EXPANSIVE allowed                                       │
│  Admin/Criminal → RESTRICTIVE only                                   │
│                                                                      │
│  PRESUMPTIONS                                                        │
│  ───────────                                                         │
│  Legal: Required by LAW (e.g., paternity 300 days)                   │
│  Factual: Based on LOGIC (smoking gun)                               │
│                                                                      │
│  KEY PRINCIPLES                                                      │
│  ─────────────                                                       │
│  Lex retro non agit = No retroactivity                               │
│  Lex superior = Higher law wins                                      │
│  Lex posterior = Later law wins                                      │
│  Lex specialis = Specific beats general                              │
│                                                                      │
│  FRENCH INFLUENCE → Contract/Private Law (not Admin)                 │
└─────────────────────────────────────────────────────────────────────┘
```
